# CNN Image Classification using CIFAR-10

## Overview
This project implements a Convolutional Neural Network (CNN) using TensorFlow to classify images from the CIFAR-10 dataset into 10 categories.

## Dataset
- CIFAR-10
- 60,000 color images (32x32)
- 10 classes
- 50,000 training images
- 10,000 testing images

## Model Architecture
- 3 Convolutional Layers
- Batch Normalization
- MaxPooling
- Dense Layers
- Dropout (0.5)

## Performance
- Final Test Accuracy: ~72%
- Evaluated using Precision, Recall, and F1-Score

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Scikit-learn

## How to Run

Install dependencies:
pip install -r requirements.txt

Run training:
python train.py
